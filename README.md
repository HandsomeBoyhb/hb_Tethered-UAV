This is the ground box controller code for a tethered lighting unmanned aerial vehicle (with rich interfaces and strong scalability). The controller is developed based on GD32F103C8T6 (FreeRTOS) and has OLED display, RS485 communication with the unmanned aerial vehicle, key control, and low power consumption functions. The controller is fully functional, allowing control of the unmanned aerial vehicle's takeoff, ascent, descent, yawing in the horizontal direction, and pitch in the horizontal direction through keys. At the same time, the corresponding control action information is displayed on the OLED screen. The brightness of the lighting can be adjusted through the keys. The OLED screen displays GPS signal strength, unmanned aerial vehicle flight altitude, remote control lock flag, and remote control connection status. The controller also has an autonomous line retraction function, completing the retraction and extension of the line device motor based on the different line tension forces of the unmanned aerial vehicle during different flight states.
